# مستندات واجهة برمجة تطبيقات المحادثات (Chat API)

الرابط الأساسي (Base URL): `/api/v1/chat`

هذا الملف يوضح بالتفصيل مسارات (Endpoints) الخاصة بنظام المحادثات بين المستخدمين، كبدء محادثة جديدة، إرسال رسائل، وعرض قائمة المحادثات والرسائل.

جميع المسارات أدناه **محمية (Protected)** وتتطلب إرسال التوكن (Bearer Token) في الترويسة (Headers).

---

## 1. بدء محادثة أو الحصول على محادثة سابقة
يبحث هذا المسار عن محادثة سابقة حول إعلان معين بين المستخدم الحالي وصاحب الإعلان. إذا لم توجد محادثة مسبقة، يقوم بإنشاء واحدة جديدة.
> **ملاحظة:** لا يمكن للمستخدم بدء محادثة مع نفسه. كما يمنع بدء محادثة إذا كان أحد الطرفين قد حظر الآخر.

- **المسار:** `POST /chat/start`
- **الحماية:** محمي (Protected)

### البيانات المطلوبة (Request Body - JSON)
| الحقل | الوصف | قواعد التحقق (Validation) |
| :--- | :--- | :--- |
| `ad_id` | معرف الإعلان | مطلوب (يجب أن يكون موجوداً في جدول الإعلانات) |

### الاستجابة الناجحة (Success Response - 200 OK)
```json
{
  "success": true,
  "message": "data_retrieved",
  "data": {
    "id": 1,
    "ad_id": 5,
    "buyer_id": 10,
    "seller_id": 2,
    "created_at": "2026-03-01T10:00:00.000000Z",
    "updated_at": "2026-03-01T10:00:00.000000Z",
    "messages": [] // مصفوفة الرسائل إن وجدت
  }
}
```

### استجابات الخطأ المحتملة (Error Responses)
- **400 Bad Request:** `cannot_message_self` (لا يمكن مراسلة نفسك).
- **403 Forbidden:** `blocked_interaction` (المراسلة محظورة بين الطرفين).

---

## 2. إرسال رسالة (Send Message)
إرسال رسالة نصية ضمن محادثة معينة. سيقوم النظام تلقائياً بإرسال إشعار (Push Notification) للطرف الآخر.

- **المسار:** `POST /chat/messages`
- **الحماية:** محمي (Protected)

### البيانات المطلوبة (Request Body - JSON)
| الحقل | الوصف | قواعد التحقق (Validation) |
| :--- | :--- | :--- |
| `conversation_id` | معرف المحادثة | مطلوب (يجب أن يكون موجوداً) |
| `content` | نص الرسالة | مطلوب (نص، حد أقصى 1000 حرف) |

### الاستجابة الناجحة (Success Response - 201 Created)
```json
{
  "success": true,
  "message": "message_sent",
  "data": {
    "id": 50,
    "conversation_id": 1,
    "sender_id": 10,
    "content": "مرحباً، هل الإعلان لا يزال متاحاً؟",
    "is_read": false,
    "created_at": "2026-03-01T10:05:00.000000Z"
  }
}
```

### استجابات الخطأ المحتملة (Error Responses)
- **403 Forbidden:** `blocked_interaction` (المراسلة محظورة بين الطرفين) أو ليس لديك صلاحية الوصول للمحادثة.

---

## 3. قائمة المحادثات الخاصة بي (My Conversations)
إرجاع قائمة بجميع المحادثات التي شارك فيها المستخدم كبائع أو كمشتري. القائمة مقسمة لصفحات (Paginated) ومرتبة تنازلياً حسب آخر رسالة مضافة (`updated_at`).

- **المسار:** `GET /chat/conversations`
- **الحماية:** محمي (Protected)

### ميزات الاستجابة:
- تتضمن بيانات الإعلان المختصرة.
- تتضمن بيانات الطرف الآخر (البائع أو المشتري) مع متوسط التقييم.
- تتضمن آخر رسالة (لتظهر في قائمة المحادثات).
- تتضمن عدد الرسائل غير المقروءة (`unread_count`).
- تدعم تقسيم الصفحات (أضف `?page=2` في الرابط للصفحة التالية).

### الاستجابة الناجحة (Paginated Response)
```json
{
  "success": true,
  "message": "data_retrieved",
  "data": [
    {
      "id": 1,
      "ad_id": 5,
      ...
      "unread_count": 2,
      "ad": { ... },
      "buyer": { ... },
      "seller": { ... },
      "messages": [
        {
          "content": "آخر رسالة في المحادثة",
          "created_at": "..."
        }
      ]
    }
  ],
  "links": { ... },
  "meta": {
    "current_page": 1,
    "last_page": 3,
    "per_page": 15,
    "total": 45
  }
}
```

## 4. عرض رسائل محادثة معينة (Get Messages)
جلب الرسائل الخاصة بمحادثة معينة.
> **سلوك مهم للمطور (Frontend/Flutter):**
> - الرسائل تأتي **مقلوبة** (الأحدث أولاً - Reverse Chronological Order) لتسهيل 구현 (Infinite Scroll).
> - يجب على واجهة المستخدم (UI) عكس المصفوفة لعرض الرسائل القديمة في الأعلى والجديدة في الأسفل.
> - عند التمرير للأعلى، يتم طلب الصفحة التالية `?page=2`.
> - بمجرد طلب هذا المسار، يتم تلقائياً تحديث حالة جميع الرسائل الواردة إلى **مقروءة** (`is_read: true`).

- **المسار:** `GET /chat/conversations/{conversationId}/messages`
- **المتغيرات (URL Params):** `conversationId` (معرف المحادثة)
- **الحماية:** محمي (Protected)

### الاستجابة الناجحة (Paginated Response - أحدث رسالة أولاً)
```json
{
  "success": true,
  "message": "data_retrieved",
  "data": [
    {
      "id": 51,
      "conversation_id": 1,
      "content": "نعم لا يزال متاحاً",
      "is_read": true,
      "created_at": "2026-03-01T10:06:00.000000Z",
      "sender": {
        "id": 2,
        "name": "البائع",
        "avatar": "..."
      }
    },
    {
      "id": 50,
      "conversation_id": 1,
      "content": "مرحباً، هل الإعلان لا يزال متاحاً؟",
      "is_read": true,
      "created_at": "2026-03-01T10:05:00.000000Z",
      "sender": { ... }
    }
  ],
  "links": { ... },
  "meta": { "current_page": 1, "per_page": 20, "last_page": 5, "total": 100 }
}
```

### استجابات الخطأ المحتملة (Error Responses)
- **403 Forbidden:** `blocked_interaction` (المراسلة محظورة) أو ليس المشتركين في هذه المحادثة.
- **404 Not Found:** المحادثة غير موجودة.
